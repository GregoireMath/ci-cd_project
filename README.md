# ci-cd_project

Requirements

python3     --> sudo apt install python3
pip3        --> python -m pip3 install --upgrade pip
flask       --> pip3 install flask
requests    --> pip3 install requests

Setup

git clone git@github.com:GregoireMath/ci-cd_project.git
cd ci-cd_project
python3 app.py

Website access

http://127.0.0.1:5000
